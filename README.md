## Vai tro

- Giong nhu "Bac bao ve"

Nha =====> Bac bao ve (middleware): Su kien (soat ve)
Nha <===== Su kien

1. Soat ve (kiem soat -> Validation)
2. Khong cho vao
3. Cho phep vao (Validation passed -> cho vao)
4. Chinh sua / Thay doi
